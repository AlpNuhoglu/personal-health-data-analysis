Project Overview

This project focuses on analyzing personal health data including step counts, calorie intake, and gym attendance patterns over a monitored period. The goal is to extract meaningful insights through detailed exploratory data analysis and machine learning techniques.

Data Collection

The data was collected from multiple health tracking sources, including:

  • Step Count: Retrieved from Apple Health App
  • Calorie Intake: Manually logged using MyFitnessPal
  • Gym Attendance: Recorded manually with binary values ('Yes' or 'No')

Data was collected daily with proper timestamps for temporal analysis.

Data Processing

Data cleaning and transformation steps were performed, including:

  • Handling missing values using forward fill techniques.
  • Standardizing numerical features.
  • Filtering outliers using the IQR method.
  • Converting categorical features to binary formats for modeling.

Exploratory Data Analysis (EDA)

Key statistical summaries and relationships were explored through:

  • Univariate Analysis: Distribution of step counts and calorie intake.
  • Bivariate Analysis: Correlation heatmaps between step counts, calorie intake, and gym attendance.
  • Outlier Detection: Identified through boxplots and Z-scores.
  • Time Series Analysis: Trends observed in daily metrics over time.

Data Visualizations

The following visualizations were generated:

  • Line charts showing daily step counts and calorie intake.
  • Scatter plots representing correlations between variables.
  • Interactive step filter plots using Altair.
  • Bar charts highlighting gym attendance frequency.

These visualizations were created using Matplotlib, Seaborn, and Altair for interactive graphs.

Modeling & Machine Learning

Multiple models were implemented to predict health trends:

  • Multiple Linear Regression: To predict weight fluctuations.
  • Decision Trees: To identify factors contributing to weight loss.
  • Random Forest Regressor: For more robust and nonlinear modeling.
  • K-Means Clustering: To segment health behavior patterns.

Performance metrics such as R-squared and Mean Squared Error (MSE) were evaluated.

Conclusion & Future Work

The analysis successfully demonstrated relationships between step counts, calorie intake, and gym attendance. Future work includes:

  • Expanding the dataset for longitudinal analysis.
  • Adding more health metrics such as sleep quality and heart rate.
  • Implementing deep learning models for time-series forecasting.
